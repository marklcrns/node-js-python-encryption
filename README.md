# AES-256-CBC Encryption/Decryption Nodejs and Python

## Dependencies

### Nodejs

- Uses nodejs built-in `crypto` library
- `npm install urlsafe-base64` - for base64 encoding

### Python

- `pip install pycryptodome`

## Run

### Nodejs

- `node cypher_b64.js`
- `node cypher_hex.js`

### Python

- `./cypher_b64.py`
- `./cypher_hex.py`
