# AES-256-CBC Encryption/Decryption Nodejs and Python

## Dependencies

### Nodejs

- Uses nodejs built-in `crypto` library
- `npm install urlsafe-base64` - for base64 encoding

### Python

- `pip install pycryptodome`

## Run

### Nodejs

- `./cypher_b64.js` - base64
- `./cypher_hex.js` - hex

### Python

- `./cypher_b64.py` - base64
- `./cypher_hex.py` - hex
